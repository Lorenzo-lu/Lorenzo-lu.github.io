General Score Function Estimator Method

In neural networks, gradient is significant in optimization. Most simple mathematical functions such as activation functions and convolution have definite derivative while random functions which sample data from some pre-determined distributions usually don’t. Therefore, the determination of their gradient is difficult. The score function estimator is a common solution to this question.

Principle of score function estimator

Suppose zzz is the random variable obeying some distribution (such as Poisson), and we want to estimate the gradient of f(z)f(z)f(z) with respect to zzz. It is intuitive to use ∇E[f(z)]\nabla\text{E}[f(z)]∇E[f(z)] instead.

PyTorch code

import numpy as np
import torch
import torch.nn as nn

class ScoreFunctionLayer(nn.Module):
   def __init__(self, distribution):
       super().__init__();
       self.distribution = distribution;
   def forward(self, *args, **kwargs):
       dist = self.distribution(*args, **kwargs);
       z = dist.sample();
       log_pz = dist.log_prob(z);
       z_log_pz = log_pz * z.data;
       out = (z - z_log_pz).data + z_log_pz; 
       return out

How PyTorch works

PyTorch uses tensor as the basic data type. There is an attribute requires_grad in tensors and their default value is False. The backpropagation in PyTorch computes the gradient of each layer based on the gradient from the next layer and current layer. If requires_grad is False, this tensor will be treated as a constant. Here is an example, y=cx\mathbf{y} = c\mathbf{x}y=cx. In PyTorch, the tensor y and x has requires_grad as True while the tensor c has that attribute False. In backpropagation, the gradient of this layer will just be c. Then this gradient will be assigned to x.grad for later optimization.

Poisson and Gaussian random implementation

This module for Poisson random layer has been verified. To call Poisson layer,

Poisson = ScoreFunctionLayer(torch.distributions.poisson.Poisson);
x = torch.tensor([10. for _ in range(10)], requires_grad=True); 
out = Poisson(x);

To test if it also works for Gaussian, we can test a normal distribution N(μ,μ)\mathcal{N}(\mu, \sqrt{\mu})N(μ,μ​). This special distribution has the mean equal to the variance.
