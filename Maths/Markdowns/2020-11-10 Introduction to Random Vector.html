  7. Outer product of vectors. Hold the same definition for x,y\mathbf{x}, \mathbf{y}x,y, their outer product ∣x⟩⟨y∣=xy⊤|\mathbf{x}\rangle \langle \mathbf{y}| = \mathbf{x} \mathbf{y}^\top∣x⟩⟨y∣=xy⊤. Random vector and random matrix

Suppose X=[X1,X2,...]⊤\mathbf{X} = [X_1, X_2, ...]^\topX=[X1​,X2​,...]⊤ and each {Xi}\{ X_i \}{Xi​} is a random variable, then X\mathbf{X}X is a random vector. Similarly, suppose M={X1,X2,...}\mathbf{M} = \{ \mathbf{X_1}, \mathbf{X_2}, ... \}M={X1​,X2​,...} and each of {Xi}\{\mathbf{X_i}\}{Xi​} is a random vector, then M\mathbf{M}M is a random matrix.

*Expectation

E[X]=[E[X1],E[X2],...]⊤\text{E}[\mathbf{X}] = [\text{E}[X_1], \text{E}[X_2], ...]^\topE[X]=[E[X1​],E[X2​],...]⊤ is also called as mean vector.

If M=[X11⋯X1c⋮⋱⋮Xr1⋯Xrc]\mathbf{M} = \left[\begin{matrix} X_{11} &\cdots &X_{1c} \\ \vdots &\ddots &\vdots\\ X_{r1} &\cdots &X_{rc}\end{matrix}\right]M=⎣⎢⎢⎡​X11​⋮Xr1​​⋯⋱⋯​X1c​⋮Xrc​​⎦⎥⎥⎤​, then E[M]=[E[X11]⋯E[X1c]⋮⋱⋮E[Xr1]⋯E[Xrc]]\text{E}[\mathbf{M}] = \left[\begin{matrix} \text{E}[X_{11}] &\cdots &\text{E}[X_{1c}] \\ \vdots &\ddots &\vdots\\ \text{E}[X_{r1}] &\cdots &\text{E}[X_{rc}]\end{matrix}\right]E[M]=⎣⎢⎢⎡​E[X11​]⋮E[Xr1​]​⋯⋱⋯​E[X1c​]⋮E[Xrc​]​⎦⎥⎥⎤​.

Suppose A\mathbf{A}A and M\mathbf{M}M has the same dimensions and A\mathbf{A}A has no random entries, then E[(AM)ij]=∑kE[aikXkj]=∑kaikE[Xkj]=(AE[M])ij\text{E}[(\mathbf{AM})_{ij}] = \sum_k \text{E}[a_{ik}X_{kj}] = \sum_k a_{ik}\text{E}[X_{kj}] = (\mathbf{A}\text{E}[\mathbf{M}])_{ij}E[(AM)ij​]=∑k​E[aik​Xkj​]=∑k​aik​E[Xkj​]=(AE[M])ij​.

*Correlation

Suppose X\mathbf{X}X is a n-dimensional random vector, and E[X]\text{E}[\mathbf{X}]E[X] is its expectation, the correlation matrix is defined by R=E[XX⊤]\mathbf{R} = \text{E}[\mathbf{X}\mathbf{X}^\top]R=E[XX⊤]. The element rij=E[XiXj]r_{ij} = \text{E}[X_i X_j]rij​=E[Xi​Xj​]. Since Xi,XjX_i, X_jXi​,Xj​ are random variable, their order does not matter. Thus rij=E[XiXj]=E[XjXi]=rjir_{ij} = \text{E}[X_iX_j] = \text{E}[X_jX_i] = r_{ji}rij​=E[Xi​Xj​]=E[Xj​Xi​]=rji​, and R=R⊤\mathbf{R} = \mathbf{R}^\topR=R⊤.

The Cauchy-Schwartz Inequality for scalar random variables is ∣E[UV]∣≤E[U2]E[V2]|\text{E}[UV]| \le \sqrt{\text{E}[U^2]\text{E}[V^2]}∣E[UV]∣≤E[U2]E[V2]​.

Proof: 0≤E[(U−λV)2]=E[U2]−2λE[UV]+λ2E[V2]0 \le \text{E}[(U - \lambda V)^2] = \text{E}[U^2] - 2\lambda \text{E}[UV] + \lambda^2 \text{E}[V^2]0≤E[(U−λV)2]=E[U2]−2λE[UV]+λ2E[V2]. Put λ=E[UV]E[V2]\lambda = \frac{\text{E}[UV]}{\text{E}[V^2]}λ=E[V2]E[UV]​:

*Covariance

Suppose X\mathbf{X}X is a n-dimensional random vector, and E[X]\text{E}[\mathbf{X}]E[X] is its expectation, the covariance matrix is defined by cov(X)=E[(X−E[X])(X−E[X])⊤]\text{cov}(\mathbf{X}) = \text{E}[(\mathbf{X} - \text{E}[\mathbf{X}])(\mathbf{X} - \text{E}[\mathbf{X}])^\top]cov(X)=E[(X−E[X])(X−E[X])⊤], which contains nnn rows and nnn columns.

Notating the covariance matrix with C\mathbf{C}C.

  1. Cii=cov(Xi,Xi)=Var[Xi]\mathbf{C}_{ii} = \text{cov}(X_i, X_i) = \text{Var}[X_i]Cii​=cov(Xi​,Xi​)=Var[Xi​].
  2. Cij=E[(Xi−E[Xi])(Xj−E[Xj])=E[(Xj−E[Xj])(Xi−E[Xi])=Cji\mathbf{C}_{ij} = \text{E}[(X_i - \text{E}[X_i])(X_j - \text{E}[X_j]) = \text{E}[(X_j - \text{E}[X_j])(X_i - \text{E}[X_i]) = \mathbf{C}_{ji}Cij​=E[(Xi​−E[Xi​])(Xj​−E[Xj​])=E[(Xj​−E[Xj​])(Xi​−E[Xi​])=Cji​. In the other word, C\mathbf{C}C is symmetric.
  3. If Xi,XjX_i, X_jXi​,Xj​ are uncorrelated, Cij=0\mathbf{C}_{ij} = 0Cij​=0. So, if Xi,XjX_i, X_jXi​,Xj​ are uncorrelated for any i≠ji \ne ji​=j, C\mathbf{C}C is a diagonal matrix.

For a symmetric matrix C\mathbf{C}C, if for any vector x\mathbf{x}x, x⊤Cx≥0\mathbf{x}^\top \mathbf{C} \mathbf{x} \ge 0x⊤Cx≥0, C\mathbf{C}C is positive semidefinite. It is positive definite iff x⊤Cx>0\mathbf{x}^\top \mathbf{C} \mathbf{x} > 0x⊤Cx>0.

*Linear estimation of random vectors (Wiener filters)

Suppose X\mathbf{X}X, Y\mathbf{Y}Y are random vectors, and only Y\mathbf{Y}Y is observed. So we need to estimate X\mathbf{X}X.

Suppose X~=g^(Y)\tilde{\mathbf{X}} = \hat{g}(\mathbf{Y})X~=g^​(Y) is the estimate or "guess" of X\mathbf{X}X. We define the best g^\hat{g}g^​ by minimizing the Mean-Squared Error (MSE), E[∥X−g^(Y)∥2]\text{E}[\| \mathbf{X} - \hat{g}(\mathbf{Y}) \|^2]E[∥X−g^​(Y)∥2]. If g^(Y)=AY+b\hat{g}(\mathbf{Y}) = \mathbf{AY} + \mathbf{b}g^​(Y)=AY+b, this form is called as affine. If b=0\mathbf{b} = 0b=0, g^\hat{g}g^​ is linear. The optimal g^\hat{g}g^​ is called as linear minimum mean-squared-error estimator or MMSE estimator or Wiener filters sometimes.

MSE=E[∥X−(AY+b)∥2]=E[∥(X−E[X]−AY+AE[Y])+(E[X]−AE[Y]−b)∥2]=E[(X−E[X]−AY+AE[Y])2]+E[(E[X]−AE[Y]−b)2]+2(E[X]−AE[Y]−b)E[(X−E[X]−AY+AE[Y])]\text{MSE} = \text{E}[\| \mathbf{X} - (\mathbf{AY} + \mathbf{b}) \|^2] = \text{E}[\| (\mathbf{X} - \text{E}[\mathbf{X}] - \mathbf{AY} + \mathbf{A}\text{E}[\mathbf{Y}]) + ( \text{E}[\mathbf{X}] - \mathbf{A}\text{E}[\mathbf{Y}] - \mathbf{b})\|^2 ] = \text{E}[(\mathbf{X} - \text{E}[\mathbf{X}] - \mathbf{AY} + \mathbf{A}\text{E}[\mathbf{Y}])^2] + \text{E}[( \text{E}[\mathbf{X}] - \mathbf{A}\text{E}[\mathbf{Y}] - \mathbf{b})^2] + 2 ( \text{E}[\mathbf{X}] - \mathbf{A}\text{E}[\mathbf{Y}] - \mathbf{b}) \text{E}[(\mathbf{X} - \text{E}[\mathbf{X}] - \mathbf{AY} + \mathbf{A}\text{E}[\mathbf{Y}])]MSE=E[∥X−(AY+b)∥2]=E[∥(X−E[X]−AY+AE[Y])+(E[X]−AE[Y]−b)∥2]=E[(X−E[X]−AY+AE[Y])2]+E[(E[X]−AE[Y]−b)2]+2(E[X]−AE[Y]−b)E[(X−E[X]−AY+AE[Y])]. Since E[(X−E[X]−AY+AE[Y])]=0\text{E}[(\mathbf{X} - \text{E}[\mathbf{X}] - \mathbf{AY} + \mathbf{A}\text{E}[\mathbf{Y}])] = 0E[(X−E[X]−AY+AE[Y])]=0, MSE=E[(X−E[X]−A(Y−E[Y]))2]+E[(E[X]−AE[Y]−b)2]\text{MSE} = \text{E}[(\mathbf{X} - \text{E}[\mathbf{X}] - \mathbf{A(Y - \text{E}[\mathbf{Y}])})^2] + \text{E}[( \text{E}[\mathbf{X}] - \mathbf{A}\text{E}[\mathbf{Y}] - \mathbf{b})^2]MSE=E[(X−E[X]−A(Y−E[Y]))2]+E[(E[X]−AE[Y]−b)2].

The minimal for the right part is achieved by putting b=E[X]−AE[Y]\mathbf{b} = \text{E}[\mathbf{X}] - \mathbf{A}\text{E}[\mathbf{Y}]b=E[X]−AE[Y]. Then the estimate is X=AY+b=A(Y−E[Y])+E[X]\mathbf{X} = \mathbf{AY} + \mathbf{b} = \boxed{ \mathbf{A(Y - \text{E}[\mathbf{Y}])} + \text{E}[\mathbf{X}] }X=AY+b=A(Y−E[Y])+E[X]​.

The minimal for the left part is achieved by putting A(Y−E[Y])=X−E[X]→AE[(Y−E[Y])(Y−E[Y])⊤]=E[(X−E[X])(Y−E[Y])⊤]→ACY=CXY\mathbf{A}(\mathbf{Y} - \text{E}[\mathbf{Y}]) = \mathbf{X} - \text{E}[\mathbf{X}] \to \mathbf{A} \text{E}[(\mathbf{Y} - \text{E}[\mathbf{Y}])(\mathbf{Y} - \text{E}[\mathbf{Y}])^\top] = \text{E}[(\mathbf{X} - \text{E}[\mathbf{X}])(\mathbf{Y} - \text{E}[\mathbf{Y}])^\top] \to \boxed{\mathbf{AC_Y} = \mathbf{C_{XY}}}A(Y−E[Y])=X−E[X]→AE[(Y−E[Y])(Y−E[Y])⊤]=E[(X−E[X])(Y−E[Y])⊤]→ACY​=CXY​​.
