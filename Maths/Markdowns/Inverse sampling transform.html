Inverse Sampling Transform

Now we have a set of random number from a uniform distribution, can we transform it into any kind of distribution we want?

Suppose our target distribution has a PMF f(x)f(x)f(x), and its CDF is F(x)=∫−∞xf(t)dtF(x) = \int_{-\infty}^x f(t) d tF(x)=∫−∞x​f(t)dt. We know F(x)∈(0,1)F(x) \in (0,1)F(x)∈(0,1).

How about the uniform distribution in (0,1)(0,1)(0,1)? Its PMF is fu(x)=1f_u(x) = 1fu​(x)=1 and its CDF is Fu(x)=xF_u(x) = xFu​(x)=x. So, if for an arbitrary yyy, Fu(x)=F(g(x))=yF_u(x) = F(g(x)) = yFu​(x)=F(g(x))=y, and then we see how to get the new random values by finding the g(x)g(x)g(x).

F(g(x))=xF(g(x)) = xF(g(x))=x, thus g(x)=F−1(x)g(x) = F^{-1}(x)g(x)=F−1(x). Now, the transform is clear. Simply generate a uniform distribution random variable xxx, and then take the F−1(x)F^{-1}(x)F−1(x).

If we don't know the expressions for fff or FFF, but only know some numerical information, we can use interpolation. A typical application: define a "histeq" function in MATLAB referring another image.

% matlab code for inverse sampling tranform 
% the following example is to generate the exp(1/5) by uniform distribution
% feel free to change the lambda;
clear
close all
clc
%% direct 
global lambda;
lambda = 1/5;
% mean is 1/lambda, for matlab code
x0 = exprnd(1/lambda, [1,100]);
figure();
histogram(x0);
title('direct');
sample_mean = mean(x0);
fprintf("The parameter lambda by direct sampling is %.4e", 1/sample_mean)
%% inverse sampling
x1 = rand(1,100);
x2 = [0];
for i = 1:length(x1)
    x2(end+1) = NSolve(x1(i));
end
x2 = x2(2:end);

figure();
histogram(x2)
title('inverse')
sample_mean = mean(x2);

fprintf("The parameter lambda by inverse sampling is %.4e", 1/sample_mean)

function [x2] = NSolve(x1)
    global lambda;
    syms x
    x2 = vpasolve(1- exp(-(lambda)*x) == x1, x);
end
